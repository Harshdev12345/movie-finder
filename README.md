# Movie Finder App 🎬

A simple movie search website built using HTML, CSS and JavaScript.

## Features
- Search movies by name
- Fetches data from OMDb API
- Displays movie posters and release year

## Technologies
- HTML
- CSS
- JavaScript
- OMDb API

## How to Run
1. Clone the repo
2. Open index.html
3. Enter a movie name and search
